# プロンプト例：PS-15 ボタン箱（上下同一ハーフ設計）

似た箱を依頼する際、以下の粒度で指示すると一発で設計に入れる。

## 指示プロンプト例

```
PS-15 ボタン用の箱を OpenSCAD で設計して。

【構成】
- 上下同一ハーフの2分割（同じSTL×2枚印刷・反転組立）
  成型誤差吸収のため本体+蓋ではなく対称2分割にする
- 外寸 47×47×47mm（各ハーフ23.5mm）、壁厚2.0、底板2.0mm
- 角R4（垂直エッジ）、接地面4辺R1.5フィレット（合わせ面はシャープ）

【機能】
- ボタン穴φ30（天面中央、反転時に上に来る）
  ※底板はPS-15仕様max2.5mmだが脱着しやすさ優先で2mmに
- 配線穴φ12（右壁の合わせ面・半円切り欠き×2で円形）
- 固定は対角2点M3インサート熱圧入＋M3×55ボタンヘッドボルト
  ※4コーナーだとPS-15交換時に指が入らないので対角2点
- 位置合わせは南北壁のみ楔型ロケーティングキー2組
  （凸：接線10→8×法線2.0×H5、凹：接線10.4→8.4×法線2.0×D5.25、接線クリア0.2）
- 内側刻印は穴外の上下余白、3mm
```

## 落とし穴（仕様に必ず明記）

1. **「上下同一」は反転組立** — 楔の凸凹座標は反転後噛み合うよう X/Y を振り分ける
   - `rotate([180,0,180]) translate([OUTER_W, 0, 2*HALF_H])` で上ハーフを重ねる
   - 凸↔凹の位置は同一ハーフ内で X方向に振り分け（例：凸X=W/4, 凹X=3W/4）

2. **合わせ面はシャープ保持** — 底面フィレットは minkowski+sphere後に cube で intersection 上端クリップ
   - 全方向 minkowski だと合わせ面も丸まって密着しない

3. **楔は南北のみ** — 4辺全部だと X/Y 両方向で過剰拘束、FDM公差で組めない
   - 南北（前後壁）で Y方向の位置決め、X方向は対角2点M3ボルトで拘束
   - 役割分担させる

4. **凹にもテーパー** — 凸と同方向のテーパーで位置決めキー効果
   - 凸 10→8mm（底→先端）、凹 10.4→8.4mm（開口→奥）
   - 接線方向のクリアが一定0.2mm ＝ 無理なく自動センタリング

5. **固定穴は対角2点（4コーナーはダメ）**
   - 4点だとPS-15交換時に内部に指が入らない
   - 対角の空きコーナーに指を入れてボタンを外す

6. **底板厚とカウンターボアの関係**
   - BOT_T=2mm、CBORE_H=1mm → cbore残し1mm（ボルト頭1.65mmのうち0.65mm突出、ゴム足で吸収）
   - BOT_T を減らす時は CBORE_H も減らして残し≥1mm 確保

7. **ボルト長計算**
   - cbore残し(1mm) + 上内高(21.5) + 下内高(21.5) = 44mm
   - +インサートL(8mm) = 52mm → 5mm切上で M3×55

## 設計上の派生パラメータ

- INNER_CORNER_R = CORNER_R - WALL = 4 - 2.0 = 2.0mm
- PILOT_D = INSERT_OD + 0.1 = 4.7mm
- BOSS_H = PILOT_DEPTH - BOT_T = 6.5mm
- SOCK_H = WEDGE_H + 0.25 = 5.25mm（逃げ）
- WEDGE_NORM = WALL（楔法線方向は壁厚いっぱい＝貫通）
